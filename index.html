i4GxZuri Internship

A backend-focused internship run in partnership between Ingressive for Good and Zuri. Over four stages, interns built real-world Node.js projects covering server-side rendering, REST APIs, and terminal applications.

Backend · Node.js · 4 Stages

Projects

Stage 1 — Portfolio Website

A simple personal portfolio built with Node.js. Pages are served server-side using routes and controllers — no frontend framework, just clean Express routing.

Stage 2 — Flight Booking REST API

A RESTful API for booking flights. Supports creating, reading, updating, and deleting flight bookings. Built with Node.js and Express, following standard REST conventions.

View on GitHub

Stage 3 — Terminal Guessing Game

A number guessing game that runs directly in the Node.js terminal. The program generates a random number and prompts the user to guess it, giving hints until they get it right.

View on GitHub

Stage 4 — Todo REST API

A lightweight REST API for managing todos, using in-memory arrays as the data store — no database. Covers full CRUD operations and demonstrates core API design without infrastructure overhead.
